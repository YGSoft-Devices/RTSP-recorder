## ‚ùì Frequently Asked Questions

**Q: Where do I start?**  
A: See "Start Here" section above based on your role.

**Q: How do I register my device?**  
A: Follow [QUICK_START.md](QUICK_START.md) ‚Üí 5-Minute Quick Start

**Q: What are the API endpoints?**  
A: See [DEVICE_INTEGRATION.md](DEVICE_INTEGRATION.md) ‚Üí API Integration

**Q: How long does registration take?**  
A: ~2 minutes (including obtaining token from admin)

**Q: Can I re-use the token?**  
A: No. Tokens are one-time use only (burned after first registration)

**Q: What if I get an error?**  
A: Check [QUICK_START.md](QUICK_START.md) ‚Üí Troubleshooting

**Q: Where are logs stored?**  
A: `~/.updates-manager-tool/logs/app.log`

**Q: Can I disable heartbeat?**  
A: Heartbeat runs automatically after registration. You can unregister to stop it.

**Q: How often does heartbeat send data?**  
A: Every 60 seconds (default, configurable)

**Q: Is my data secure?**  
A: Yes. See [MEETING_DEVICE_INTEGRATION.md](MEETING_DEVICE_INTEGRATION.md) ‚Üí Security Considerations

**Q: What if the server is unreachable?**  
A: Automatic retry (up to 5 attempts). See error handling in [DEVICE_INTEGRATION.md](DEVICE_INTEGRATION.md)
